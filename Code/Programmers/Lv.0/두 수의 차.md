### 문제 설명
정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.

### 제한사항
-50000 ≤ num1 ≤ 50000\
-50000 ≤ num2 ≤ 50000

```python
def solution(num1, num2):
    answer = num1 - num2
    return answer

print(solution(2,-3))
```
