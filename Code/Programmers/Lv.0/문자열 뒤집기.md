# 문자열 뒤집기
https://school.programmers.co.kr/learn/courses/30/lessons/120822

### 문제 설명
문자열 my_string이 매개변수로 주어집니다. my_string을 거꾸로 뒤집은 문자열을 return하도록 solution 함수를 완성해주세요.

### 코드
```python
def solution(my_string):
    answer = ''
    for i in range(len(my_string)-1,-1,-1):
        answer += my_string[i]
    return str(answer)
```

### 노트
