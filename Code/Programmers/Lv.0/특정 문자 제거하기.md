# 특정 문자 제거하기
https://school.programmers.co.kr/learn/courses/30/lessons/120826

### 문제
자 제거하기
문제 설명
문자열 my_string과 문자 letter이 매개변수로 주어집니다. my_string에서 letter를 제거한 문자열을 return하도록 solution 함수를 완성해주세요.

### 코드
```python
def solution(my_string, letter):
    answer = ''
    for ms in my_string:
        if (ms!=letter):
            answer+=ms
    return answer
```
